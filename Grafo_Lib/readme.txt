1) COMO RODAR O PROGRAMA:

	Caso queira:

---->Compilar os testes para a lista simples, digite "make lista_simples_teste".
---->Compilar os testes para a lista duplamente encadeada, digite "make lista_dupla_teste".
---->Compilar os testes para o grafo, digite "make grafo_teste".
---->Compilar o módulo principal que realiza operações sobre o grafo, digite "make main".
---->Compilar tudo gerando um programa principal, digite "make all".
---->Limpar o dirtório de arquivos objeto, digite "make clean".

	Os arquivos executáveis gerados possuem o mesmo nome da regra digitada.
